Lesson 1: how to search for a node

  • Match any node and give it the name N
  • Impose some restriction on the feature structure
  • Use quotes around values if they contain special characters
  • Impose several restrictions on the feature structure
  • Use disjunction on feature values
  • Use negation on feature values
  • Require a feature to be there (without restriction on the value)
  • Require a feature not to be there

Lesson 2: how to search for a relation

  • Implicit node declaration
  • Search for a given relation
  • Search for a disjunction of relations
  • Search for a relation different from a given disjunction
  • Use regular expression to match a set of label
  • Combine restriction on nodes and restriction on relations
  • Search for a larger pattern
