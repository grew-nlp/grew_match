Lesson 1: how to search for a node

  • Match any node and give it the name N
  • Impose some restriction on the feature structure
  • Use quotes around values if they contain special characters
  • Impose several restrictions on the feature structure
  • Use disjunction on feature values
  • Use negation on feature values
  • Regular expression in feature value
  • Require a feature to be there (without restriction on the value)
  • Require a feature not to be there
  • Matching is injective

Lesson 2: how to search for a relation

  • Search a relation without restriction
  • Implicit node declaration
  • Search for a given relation
  • Search for a disjunction of relations
  • Search for a relation different from a given disjunction
  • Use regular expression to match a set of label
  • Combine restriction on nodes and restriction on relations
  • Search for a larger pattern

Lesson 3: add constraints on patterns

  • Constraint for the equality of two features
  • Constraint for the feat_inequality of two features
  • Constraint on positions in the sentence
  • Immediate precedence
  • Constraint on incoming edges
  • Constraint on outgoing edges

Lesson 4: use negative application patterns

  • Filter out results of the pattern part
  • Remove occurrences with a given relation
  • one or two negative application patterns [1/2]
  • one or two negative application patterns [2/2]
  • multiple negative pattern for error mining

Lesson 5: clustering of the results

  • Cluster by "upos" for a given form
  • Cluster results by one node "upos"
  • Cluster results by one node "lemma"
  • Other clustering on nodes
  • Cluster results by edge label
